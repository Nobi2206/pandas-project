 Paint Product Data Analysis using Pandas

 Project Overview
This project focuses on cleaning and analyzing paint product data using Python Pandas.
The dataset contains product details such as brand, color, finish, price, volume,
VOC level, and shelf life.

 Dataset Details
Rows: 5000  
Columns: 10  

Columns included:
- Product_ID
- Brand
- Product_Name
- Color
- Finish
- Product_Type
- Volume_ml
- Price_INR
- VOC_Level
- ShelfLife_Months

Steps Performed
- Imported dataset using Pandas
- Checked and handled missing values
- Removed duplicates
- Corrected data types
- Saved cleaned dataset
- Performed basic data inspection and analysis

 Tools Used
- Python
- Pandas
- Google Colab
